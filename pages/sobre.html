Sobre o SkateHub

O SkateHub é um site criado para reunir informações sobre o mundo do skate. O objetivo é apresentar skatistas famosos, skateparks, manobras e outros conteúdos relacionados com esta modalidade.

Limitações

  • As imagens são armazenadas localmente, sem servidor ou base de dados
  • Os dados dos skatistas e skateparks são estáticos, definidos manualmente no código
  • Sem sistema de autenticação ou perfil de utilizador
  • O tema claro/escuro é guardado localmente e pode não persistir em todos os browsers

Este projeto foi desenvolvido no âmbito da unidade curricular de Tecnologias da Internet, utilizando HTML, CSS e JavaScript.
